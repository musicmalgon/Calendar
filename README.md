# Calendar

|---|---|---|---|
|일정생성|POST|/api/calendar|CalendarResponseDto|
|일정조회|GET|/api/calendar/{calendarId}|List<CalendarResponseDto>|

|일정삭제|DELETE|/api/calendar/{calendarId}|Long|
